Welcome to the GHI repository! I hope you will be loving it!

To reproduce the analyses run in the top-level directory:

```
make all
```

To reproduce the manuscripts run in the top-level directory:

```
make tex
```

System Requirements
- GNU Makefile
- git
- LaTeX and pdflatex
- R
- R packages
 - ITHIM
 - ggplot2
 - and more
